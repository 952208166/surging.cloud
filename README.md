# surging.cloud　　　　　　　　　　　　　　　　　　　　　　　　　　　　　　　　　　　　　　

## 简介
surging.cloud 是在[surging](https://github.com/dotnetcore/surging) 的基础上,基于我个人的爱好和学习,在我的理解上进行了部分调整的。主要是在服务通信过程中做了些许调整和扩展了个别的组件。
